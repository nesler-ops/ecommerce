🎸Guitar Store – E-commerce Web Application
Overview

Guitar Store is a full-stack e-commerce web application designed for browsing and purchasing musical instruments online. 
The platform provides users with an intuitive interface to explore a catalog of guitars, manage a shopping cart, and simulate a checkout process.

This project was built to demonstrate modern web development skills, including frontend and backend integration, API design, and user interaction.

Features:

🎸 Browse a catalog of guitars
🔍 Search and filter products
🛒 Add and remove items from the cart
👤 User authentication (login/register) (if implemented)
💳 Simulated checkout process
📱 Responsive design for mobile and desktop
